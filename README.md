前端安装
-------------
1. 安装node js
https://nodejs.org/dist/v16.16.0/win-x64/node.exe

2. 通过打开文件夹的cmd， 输入
```
node -v
```
查看node.js有没有装好

3. 进入terminal，_Front内 安装npm后，输入
```
npm yarn install
```

4. 输入
```
yarn install 
```
安装依赖

5. 输入
```
yarn run dev -o 
```
直接打开， 或者点击端口http://localhost:3000/
